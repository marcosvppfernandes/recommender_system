The definition of a Board Game is pretty open, but normally the rules are clear, needs a mechanic to use during the game and some sort of definition of how someone can win in that game. We can find games that uses pieces, boards, cards, dice, coins, resources and nowadays we even have eletronics involved in board games.

You can also visit the site where we got our data from, where you can get more information on the recommended game: BoardGameGeek

Recommendation Systems (also known as recommendation engines) are a relatively new prediction method that uses algorithms such as Collaborative filtering (CF) and Clustering that try to provide a number N of items (not yet interacted) for a specific user based on the correlation between users and what they had interaction with. In our case we initially try to implement a CF using a dataset with over 15 million rows.
